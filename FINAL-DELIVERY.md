## 🎯 DEPLOYMENT (5 MINUTES)

### 1. Backend Setup
```bash
cd backend
cp .env.example .env

# Edit .env - REQUIRED:
# - DB_PASSWORD
# - JWT_SECRET
# - JWT_REFRESH_SECRET
# 
# OPTIONAL (for features):
# - SLACK_BOT_TOKEN
# - AWS_ACCESS_KEY_ID / AWS_SECRET_ACCESS_KEY
# - PAGERDUTY_API_KEY
# - GITHUB_TOKEN
# - DATADOG_API_KEY
# - SMTP credentials

cd ../docker
docker-compose up -d
```

### 2. Frontend Setup
```bash
cd frontend
npm install
cp .env.local.example .env.local

# Edit .env.local:
# NEXT_PUBLIC_API_URL=http://localhost:5000
# NEXT_PUBLIC_WS_URL=ws://localhost:5000

npm run dev
```

### 3. Verify

- Frontend: http://localhost:3000
